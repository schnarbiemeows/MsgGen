RecordMakerAndCopierActor 

class RecordMakerAndCopierActor extends Actor

this is an actor class. It's purpose is to: 1. create Records using meta-data contained in a RecordsTemplate into an initial array 2. copy a Record object from an intial array to a second, double array at a given index
